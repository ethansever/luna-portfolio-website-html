Parallel Play

UX/UI Design

Parallel Play is an extensive database of culturally diverse plays and playwrights which can be search and discovered through extensive filters.

Date
July 2023 - September 2023
My role
UX/UI Designer
Team size
N/A

Problems

This UX/UI design case focuses on creating a database for plays and playwrights, while creating it my main goal was to create a database which users can smoothly navigate through. The main goals are:

  • Creating intuitive filters for the users
  • Creating hierarchy among elements, set by experts in the theatre industry
  • Designing home page, welcoming and easy to navigate for discovery
