Parallel Play

Parallel Play is an extensive database of culturally diverse plays and playwrights which can be search and discovered through extensive filters.

Figma Drive

Overview

This UX/UI experience focuses on creating a database for plays and playwrights, while creating it my main goal was to create a database which users can smoothly navigate through. The main goals are:

  • Creating intuitive filters for the users
  • Creating hierarchy among elements, set by experts in the theatre industry
  • Designing home page, welcoming and easy to navigate for discovery

My Role

The team consists of two developers, a designer, a project manager (PM) and two project coordinators. This ongoing project receives extensive support from various departments within Langara to create and maintain the database I have created personas, user flows, paper prototypes, and mockups with the insights gained from user interviews.

User Flow

Filters

The filters are organized based on the content hierarchy, prioritizing the arrangement of filters according to their significance. This structure is formed by insights gathered from theater industry professionals, ensuring that the most crucial filters are placed prominently for user convenience and relevance. And users can sort the filter results by title, playwright last name, country name and date of publication.

This user interface design encourages users to refine their search based on multiple criteria, ensuring an intuitive and responsive journey ultimately elevating the user experience on the website.
