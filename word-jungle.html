Team size

Word Jungle is a fictional word puzzle mobile game which combines creating words with crossword puzzle. The aim of this project was to create a word game with a fun and lively environment yet keep colors muted to not overwhelm the user and create more relaxed game environment.

I have used Figma for wireframing, designing and prototyping. All assets throughout the design were created by me and done in Adobe Illustrator, Photoshop and Figma.

You can see the gameplay and UI in the prototype video.
