Problems and Solutions

This case study focused on UI of the app Ravit app to upgrade the user interface. The problems and solutions are:

  • Confusing Filter and Search
  • Unclear Sectioning
  • Lack of visibility for the like feature

Problems explained

  • Confusing Filter and Search Placement: In the app it is really challenging to differentiate between the filtering options and the search functionality. This confusion hinders their ability to change preferences and make customization among posts. Because of lack of hierarchy and contrast among buttons, it is hard for the user to find this features.
  • Unclear Sectioning: The app lacks clear visual divisions between different cards between posts. Users struggle to distinguish between different posts, affecting their navigation experience.
  • Lack of Visibility for the Like Feature: The "like" feature is not displayed in the homepage even though the feature exist in the post/product pages. This results in users overlooking the feature, reducing engagement and interaction on the platform.

Solution

  • Streamlined Filter and Search: To address the confusion between filters and search, the UI was revised to clearly separate these two functions and contrast of the filter button is enhanced to showcase the button. A more prominent position was given to search bar.
  • Enhanced Sectioning: New cards were designed in the UI Kit to visually differentiate content. UI Kit is created to improve sectioning in the app using consistent design elements. Clear headings, outlined labels, and subtle dividers are added to create a structured and organized layout, aiding users in identifying different content areas.
  • Added Like Feature to Home Page: The "like" feature is added to home page to increase visibility. This function was only available in the product page so the feature was added to homepage to encourage user engagement. On the product page and the homepage, a heart icon with a primary color was added. Additionally, a brief animation and feedback is implemented upon tapping the heart, enhancing the interactive experience.

Conclusion

In summary, user experience challenges were tackled by untangling the confusion around filter and search placement, enhancing content sectioning for clarity, and boosting user engagement through the integration of the "like" feature on the homepage. This helped application to become more cohesive and engaging platform by adding user-centric design principles.
